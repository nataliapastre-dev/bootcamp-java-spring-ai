## 📖 Sobre o Projeto

Este repositório foi criado para armazenar os exercícios, desafios e projetos desenvolvidos durante o **Bootcamp NTT DATA: Backend Java com Spring AI**.

O objetivo é registrar minha evolução no desenvolvimento Backend, consolidando conhecimentos em **Java**, **Programação Orientada a Objetos (POO)**, **Spring Boot**, **APIs REST** e **Inteligência Artificial com Spring AI**.

---

## 🎯 Objetivos de Aprendizagem

- Desenvolver lógica de programação com Java
- Aprender Programação Orientada a Objetos (POO)
- Trabalhar com coleções e estruturas de dados
- Construir APIs REST com Spring Boot
- Aplicar boas práticas de desenvolvimento Backend
- Integrar aplicações com bancos de dados
- Explorar recursos de Inteligência Artificial utilizando Spring AI
- Criar projetos práticos para o portfólio
